## Topics Learned Today

### 11. HTML Structure & Tags ✅

**Basic HTML Structure:**
```html
<!DOCTYPE html>
<html>
<head>
  <title>Website Title</title>
</head>
<body>
  <h1>Content here</h1>
</body>
</html>
```

**Key Tags:**
- `<h1>` to `<h6>` - Headings
- `<p>` - Paragraphs
- `<a>` - Links
- `<img>` - Images
- `<ul>`, `<li>` - Lists

---

### 12. CSS Styling ✅

**Change colors & layout:**
```css
body {
  background-color: black;
  color: white;
}

h1 {
  color: #FFD700;
}
```

**Styling elements:**
- `background-color` - Background
- `color` - Text color
- `font-size` - Text size
- `border-radius` - Rounded corners
- `padding` - Space inside

---

### 13. Loading Page / Splash Screen ✅

**Create welcome screen before content:**
```css
.loading {
  position: fixed;
  animation: fadeOut 3s forwards;
}

@keyframes fadeOut {
  0% { opacity: 1; }
  100% { opacity: 0; }
}
```

**How it works:**
1. Show loading for 3 seconds
2. Fade out to content
3. User sees welcome message!

---

### 14. Website Layout Structure ✅

**Professional layout:**
```
┌─ HEADER ─────────────────────┐
├─ NAVIGATION ─────────────────┤
├─ HERO / ABOUT ───────────────┤
├─ GALLERY / IMAGES ───────────┤
├─ CONTACT INFO ───────────────┤
└─ FOOTER ─────────────────────┘
```

**Best practices:**
- Clear header with title
- About/intro section
- Visual content (images)
- Contact info at bottom
- Footer with credits

---

### 15. Adding Images & Content ✅

**Add images from internet:**
```html
<img src="https://example.com/image.jpg" alt="Description">
```

**Add clickable links:**
```html
<a href="mailto:email@gmail.com">Email me</a>
<a href="https://instagram.com/username">Instagram</a>
```

**Create forms:**
```html
<input type="text" placeholder="Name">
<input type="email" placeholder="Email">
<button>Submit</button>
```
