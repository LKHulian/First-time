#### 2. **ASCII & Unicode**
- **ASCII** - Old system (128 characters)
  - Only English letters, numbers, basic symbols
  - A = 65, a = 97, @ = 64
  - Problem: Cannot display other languages or emojis

- **Unicode** - Modern standard (1+ million characters!)
  - Supports all world languages: Chinese, Arabic, Indonesian, etc.
  - Supports emojis: 😊 = 128578, 🚀 = 128640
  - Used by computers worldwide for communication
  - Different encodings: UTF-8 (most popular), UTF-16, UTF-32

**Key Concept:** Unicode allows computers to communicate across different countries and languages!

---

#### 3. **Binary to Code Journey**
Discovered how computer understands code:

**The Complete Flow:**
```
Level 1: Binary (0 and 1) - What computer stores
         ↓
Level 2: ASCII/Unicode (Numbers) - Convert binary to numbers
         ↓
Level 3: Machine Code - Numbers become instructions
         ↓
Level 4: Assembly Language - Human readable instructions
         ↓
Level 5: Programming Language (Python) - What we write!
         ↓
Back to Binary - Computer executes
```

**Example - Simple addition:**
```
You write: print(3 + 5)
    ↓
Python interprets
    ↓
Converts to machine code
    ↓
Converts to binary: 01001101 (mov), 00000011 (3), 00000101 (5), 01010011 (add)
    ↓
Computer calculates: 3 + 5 = 8
    ↓
You see: 8
```

**Key Concept:** Programming languages bridge human thinking and computer binary!

---

#### 4. **Algorithms**
- Algorithm = Step-by-step instructions for computer
- Example: When searching contacts in phone
  - **Linear Search** (slow): Check one by one (up to 500 steps)
  - **Binary Search** (fast): Smart search (only ~9 steps for 500 contacts)
- Phones use Binary Search because it's MUCH faster!

**Key Concept:** Algorithms are the "how-to" instructions that make computers work efficiently!

---

#### 5. **Why Programming Languages Exist**
- Without programming languages, we'd write everything in BINARY! 😱
- Programming languages make coding:
  - ✅ Simple and readable
  - ✅ Fast to write
  - ✅ Easy to understand
  - ✅ Hard to make mistakes

**Example:**
```
Without language: 01001000 01100101 01101100 01101100 01101111 (Hello - 5 bytes!)
With Python: print("Hello") (simple!)
```

#### 6. **Introduction to HTML**
- **HTML** = HyperText Markup Language
- HTML is the **SKELETON** of web pages 🦴
- Uses **TAGS** to structure content: `<tag> content </tag>`

**Important Tags Learned:**
- Structure: `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`
- Text: `<h1>` to `<h6>`, `<p>`, `<strong>`, `<em>`
- Lists: `<ul>`, `<ol>`, `<li>`
- Interactive: `<a>`, `<button>`, `<input>`
- Media: `<img>`

**Key Concept:** HTML tells browser WHAT to show, CSS tells HOW to style, JavaScript makes it INTERACTIVE!
